# cascad
Каскадный скрипт для проброса входящего трафика с одного сервера на другой.
Перенаправляет весь TCP/UDP трафик на удалённый сервер, за исключением портов 22, 80 и 443.
Это позволяет сохранить доступ по SSH и при необходимости использовать веб-сервер на исходной машине.
Подходит для маскировки инфраструктуры и разделения точек входа и обработки трафика.


# Рекомендация
Используете двух хостеров 
Один ру хостер, именно с рф айпи, а другой где то в Европе

# Использование скрипта 

После преобретиения серверов, устоновите пример AmeziaWG 1 на сервер в Европе, далее запутите данный скрипт из репозитория на ру сервере, и магия готова. В Конфиге который вы поличили замените айпи адресс, на айпи адресс ру хостера.

# Как это работает? 

Принцып работы довольно простой, суть такова, Ваш провайдер видит айпи адресс подключения именно к первому серверу, тоесть Рф, смысл касскада таков, что полностью весь трафик перенаправляется на дургой сервер. 
Сразу отвечу, если хостер пишет, что запрещенно использование VPN, то по сути, вы не используете его на сервере, а просто перенаправлете трафик на зарубежный сервер
